# Základní příkaz `SELECT`

``` sql
SELECT sloupec
FROM tabulka;
```

> [!tip]
> `SELECT` - Určuje, které sloupce se mají zobrazit
> `FROM` - Určuje tabulku (nebo tabulky), odkud se data berou

---

# Filtrace `WHERE`

= Podmínka pro výběr konkrétních hodnot / záznamů

``` sql
SELECT *
FROM zakaznici
WHERE mesto = 'Praha';
```

## Možné operátory s `WHERE`

|    Operátor     |          Význam          |           Příklad            |
|:---------------:|:------------------------:|:----------------------------:|
|        =        |          rovno           |      `mesto = 'Praha'`       |
|       <>        |    nerovno (jiné než)    |      `mesto <> 'Brno'`       |
|  >, <, >=, <=   |     porovnání čísel      |          `vek > 18`          |
| BETWEEN … AND … |         rozmezí          |   `vek BETWEEN 18 AND 30`    |
|     IN (…)      |      seznam hodnot       | `mesto IN ('Praha', 'Brno')` |
|      LIKE       |       vzor (text)        |      `jmeno LIKE 'P%'`       |
|     IS NULL     | test na prázdnou hodnotu |       `datum IS NULL`        |

# Řazení výsledků

``` sql
SELECT jmeno, prijmeni, vek
FROM zakaznici
ORDER BY vek DESC;
```

>[!tip]
> **ASC** – vzestupně (výchozí)
> **DESC** – sestupně

# Agregace

Slouží k zpracování hodnot ve sloupcích (počítání, sčítání, průměr atd.)

|    Funkce    |      Význam      |         Příklad          |
|:------------:|:----------------:|:------------------------:|
|   COUNT()    |  počet záznamů   |        `COUNT(*)`        |
|    SUM()     |      součet      |       `SUM(cena)`        |
|    AVG()     |      průměr      |        `AVG(vek)`        |
| MIN(), MAX() | minimum, maximum | `MIN(cena)`, `MAX(cena)` |

> [!danger] U agregace se musí použít `GROUP BY`
> To proto aby server věděl co má počítat
> Co neagregujeme, to groupujeme

``` sql
SELECT mesto, COUNT(*) AS pocet_zakazniku
FROM zakaznici
GROUP BY mesto;
```

# Spojování tabulek (`JOIN`)

Spojuje více tabulek dohromady

## Typy joinů

|    Typ JOIN    |                      Popis                      |    Schéma    |
|:--------------:|:-----------------------------------------------:|:------------:|
| **INNER JOIN** | zobrazí jen odpovídající záznamy z obou tabulek |   🔹 A ∩ B   |
| **LEFT JOIN**  |      všechny z levé + odpovídající z pravé      | 🔹 A + (A∩B) |
| **RIGHT JOIN** |      všechny z pravé + odpovídající z levé      | 🔹 B + (A∩B) |
| **FULL JOIN**  |         všechny záznamy z obou tabulek          |   🔹 A ∪ B   |

> [!tip] Pravá a levá tabulka
> Pravou a levou tabulkou se bere ta která je v tom kódu na levo nebo na pravo od `JOIN`

> [!info] Co je která tabulka
> ``` sql
> SELECT *
> FROM Zakaznici
> LEFT JOIN Objednavky
> ON Zakaznici.ID = Objednavky.ID_zakaznika;
> ```
> Zde tabulka **Zákazníci** je tabulka *levá* a **Objednávky** je tabulka *pravá*

Příklady left joinu:

> [!example]- LEFT JOIN
> = vezmi **všechny záznamy z levé tabulky** a **přidej k nim odpovídající záznamy z pravé tabulky**,  pokud existují, vyplní se `NULL` hodnotou.
>``` sql
>SELECT Zakaznici.Jmeno, Objednavky.Cena
>FROM Zakaznici
>LEFT JOIN Objednavky ON Zakaznici.ID = Objednavky.ID_zakaznika;
>```
> Výsledkem bude třeba takováto tabulka:
>
> | Zakaznici | Objednavky |
> | :---------: | :------------: |
> | Anna  | 500 Kč |
> | Petr  | NULL |
> |  Eva  | 200 Kč |

# Operátory v dotazech

| Operátor |            Význam             |                              Příklad                               |
|:--------:|:-----------------------------:|:------------------------------------------------------------------:|
|   AND    |          logické „a“          |                   `mesto = 'Praha' AND vek > 18`                   |
|    OR    |        logické „nebo“         |                `mesto = 'Praha' OR mesto = 'Brno'`                 |
|   NOT    |            negace             |                       `NOT mesto = 'Praha'`                        |
|  UNION   | spojení výsledků dvou SELECTů | `SELECT jmeno FROM zakaznici UNION SELECT jmeno FROM zamestnanci;` |

# Funkce pro práci s textem a čísly

|          Funkce          |         Význam         |            Příklad             |
|:------------------------:|:----------------------:|:------------------------------:|
|     UPPER(), LOWER()     | změna velikosti písmen |         `UPPER(jmeno)`         |
|         CONCAT()         |     spojení textů      | `CONCAT(jmeno, ' ', prijmeni)` |
|         LENGTH()         |      délka textu       |        `LENGTH(jmeno)`         |
| ROUND(), CEIL(), FLOOR() |      zaokrouhlení      |        `ROUND(cena, 2)`        |
