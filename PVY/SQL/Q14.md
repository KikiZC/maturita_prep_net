# Vkládání a modifikace dat

## Vkládání dat do tabulek

``` sql 
INSERT INTO tabulka (sloupec1, sloupec2, ...)
VALUES (hodnota1, hodnota2, ...);
```

Toto lze udělat i jinak pomocí `INSERT`

|                          Forma                          |                                     Popis                                      |                                              Příklad                                              |
|:-------------------------------------------------------:|:------------------------------------------------------------------------------:|:-------------------------------------------------------------------------------------------------:|
|       **1. INSERT – s uvedením sloupců a hodnot**       |  Nejčastější způsob – vložíme přesně definované hodnoty do vybraných sloupců   |            `INSERT INTO studenti (jmeno, prijmeni, vek) VALUES ('Petr', 'Novák', 18);`            |
|           **2. INSERT – bez uvedení sloupců**           | Musí být zadány všechny hodnoty ve stejném pořadí, jako jsou sloupce v tabulce |                  `INSERT INTO studenti VALUES (1, 'Petr', 'Novák', 18, '3.A');`                   |
|           **3. INSERT – více řádků najednou**           |                      Efektivnější při hromadném vkládání                       |       `INSERT INTO studenti (jmeno, prijmeni) VALUES ('Eva', 'Malá'), ('Lukáš', 'Dvořák');`       |
| **4. INSERT – z jiné tabulky (INSERT INTO ... SELECT)** |                           Vloží data z jiné tabulky                            | `INSERT INTO absolventi (jmeno, prijmeni) SELECT jmeno, prijmeni FROM studenti WHERE rocnik = 4;` | 

>[!info]- Chytrost systému
> Někdy lze použít formu 2 přičemž vynecháte sloupce které obsahují `AUTO_INCREMENT` nebo `DEFAULT ...`
> > [!danger] Toto funguje pouze někdy a nemusí být stoprocentně spolehlivé.

## Aktualizace dat v tabulce

``` sql
UPDATE tabulka
SET sloupec = nová_hodnota, sloupec2 = hodnota2
WHERE podmínka;
```

Příklad:
``` sql
UPDATE studenti
SET vek = 19
WHERE jmeno = 'Petr' AND prijmeni = 'Novák';
```

Tohle nastaví všem **Petrům Novákům** věk na 19.

>[!danger] **POZOR:** Pokud vynecháš `WHERE`, změní se **všechny** záznamy v tabulce!

## Mazání dat z tabulky

Toto lze udělat jednoduše pomocí `DELETE`.

``` sql
DELETE FROM tabulka WHERE podminka;
```

Příklad:
``` sql
DELETE FROM studenti WHERE rocnik = 4;
```

>[!danger] Bez `WHERE` se smažou **všechna** data v tabulce!

# Transakční režim SQL serveru

>[!Info] Transakce 
>= **jednotka práce** (soubor SQL příkazů, který se provádí jako celek).  
> Buď se provede **všechno**, nebo **nic** – zaručuje **[atomicitu](/PVY/SQL/Q11.md#ACID)**.

## Základní struktura

``` sql
BEGIN TRANSACTION;
-- nějaký kód co pracuje s daty
COMMIT;
-- nebo
ROLLBACK;
```

>[!info] Vysvětlení
>- **BEGIN TRANSACTION** – zahajuje transakci
>- **COMMIT** – potvrzuje všechny změny  
>- **ROLLBACK** – vrací databázi do stavu před začátkem transakce

## Význam uživatelských transakcí
Jakože ACID. Zde zkráceně:

| Výhoda                    | Popis                                                   |
|:------------------------- |:------------------------------------------------------- |
| **Atomicita**             | Vše se provede celé, nebo nic.                          |
| **Konzistence**           | Databáze zůstává ve správném stavu.                     |
| **Izolace**               | Změny nejsou vidět pro ostatní, dokud nejsou potvrzené. |
| **Trvalost (Durability)** | Po potvrzení (`COMMIT`) jsou změny trvale uložené.      |
