# Systémové funkce

Jsou to vestavěné funkce, které poskytuje přímo databázový systém (např. T-SQL). Slouží k rychlé manipulaci s daty přímo v dotazech.

### 1. Agregační funkce
Zpracovávají celou sadu řádků a vrací **jednu souhrnnou hodnotu**. Často se pojí s klauzulí `GROUP BY`.
- `SUM()` – součet hodnot.
- `AVG()` – průměrná hodnota.
- `COUNT()` – počet záznamů (řádků).
- `MIN()` / `MAX()` – nejmenší / největší hodnota.

### 2. Matematické funkce
Provádějí výpočty nad číselnými daty.
- `ABS()` – absolutní hodnota.
- `ROUND(hodnota, délka)` – zaokrouhlení na daný počet míst.
- `SQRT()` – druhá odmocnina.
- `POWER(základ, exponent)` – mocnina.

### 3. Funkce data a času
Práce s časovými údaji a jejich formátování.
- `GETDATE()` – vrátí aktuální datum a čas serveru.
- `DATEPART(jednotka, datum)` – vrátí konkrétní část (rok, měsíc, den) jako číslo.
- `DATEDIFF(jednotka, start, konec)` – vrátí rozdíl mezi dvěma daty.
- `DATEADD(jednotka, počet, datum)` – přičte/odečte časový úsek k datu.

### 4. Řetězcové funkce (String functions)
Práce s textem (varchary).
- `LEN()` – délka řetězce (počet znaků).
- `SUBSTRING(text, start, délka)` – vyřízne část textu.
- `UPPER()` / `LOWER()` – převede text na velká/malá písmena.
- `REPLACE(text, co, čím)` – nahradí část textu jiným textem.
- `LTRIM()` / `RTRIM()` – odstraní prázdné znaky (mezery) zleva/zprava.

---

# Uživatelské funkce (User-Defined Functions - UDF)

Uživatelské funkce si vytváří programátor pro zapouzdření opakující se logiky. 
**Hlavní pravidlo:** Funkce **musí** vždy vracet hodnotu (skalární nebo tabulku) a nesmí měnit stav databáze (uvnitř funkce nelze dělat INSERT, UPDATE, DELETE v tabulkách).

---

# Dva základní typy funkcí

| Vlastnost             | Skalární funkce                             | Tabulková funkce (TVF)             |
|:--------------------- |:------------------------------------------- |:---------------------------------- |
| **Návratová hodnota** | Jedna konkrétní hodnota (INT, VARCHAR, ...) | Celá tabulka (result set)          |
| **Použití v dotazu**  | V `SELECT`, `WHERE`, `HAVING`               | V klauzuli `FROM` (jako tabulka)   |
| **Příklad**           | Výpočet DPH k ceně                          | Seznam objednávek daného zákazníka |

---

# Vytvoření vlastní funkce

### 1. Skalární funkce (Scalar Function)
Vrací jeden výsledek.

> [!example] Vytvoření (Vrátí cenu s DPH)[^1]
> ```sql
> CREATE FUNCTION dbo.CenaSDPH (@Cena DECIMAL(10,2), @DPH DECIMAL(4,2))
> RETURNS DECIMAL(10,2)
> AS
> BEGIN
>     RETURN @Cena * (1 + @DPH)
> END
> ```

> [!abstract] Použití
> ```sql
> SELECT Nazev, dbo.CenaSDPH(Cena, 0.21) AS CenaS21DPH 
> FROM Produkty
> ```

### 2. Tabulková funkce (Table-Valued Function)
Vrací tabulku. Existují dva podtypy: **Inline** (jednoduchý SELECT) a **Multi-statement** (složitější logika v BEGIN-END). Nejčastěji se používá Inline.

> [!example] Vytvoření (Vrátí tabulku produktů v dané kategorii)[^2]
> ```sql
> CREATE FUNCTION dbo.fn_ProduktyVKategorii (@KatID INT)
> RETURNS TABLE
> AS
> RETURN (
>     SELECT ID, Nazev, Cena 
>     FROM Produkty 
>     WHERE KategorieID = @KatID
> )
> ```

> [!abstract] Použití
> ```sql
> SELECT * FROM dbo.fn_ProduktyVKategorii(5)
> WHERE Cena > 100
> ```

[^1]: Všimni si že se tam píše `dbo.`

[^2]: Taky lze použít předponu `fn_` pro zvíraznění že se jedná o funkci.
