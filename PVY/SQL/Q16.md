# Dotaz (SELECT)

- Jednorázový SQL příkaz
- Spustí se → vrátí data → **nikam se neukládá**
- Neexistuje po skončení dotazu

> [!tip] Příklad
> ``` sql
> SELECT * FROM Students WHERE Age > 18;
> ```


# Pohled (View)

- **Uložený [dotaz](#Dotaz%20(SELECT)) v databázi**
- Chová se jako **virtuální tabulka**
- Data se fyzicky neukládají (většinou), ukládá se **definice dotazu**

> [!abstract] Vytvoření
> ```sql
> CREATE VIEW AdultStudents AS
SELECT * FROM Students WHERE Age > 18;
> ```

> [!example] Použití
> ``` sql
> SELECT * FROM AdultStudents;
> ```

# Rozdíly shrnutí

|          Dotaz          |    Pohled     |
|:-----------------------:|:-------------:|
|       Jednorázový       |    Uložený    |
| Neexistuje po provedení | Existuje v DB |
| Nelze na něj dát práva  | Lze dát práva |

# Zjištění existence pohledu

``` sql
IF OBJECT_ID('dbo.AdultStudents', 'V') IS NOT NULL
    PRINT 'Pohled existuje';
```

Jednoduchá podmínka dotazující se na jméno (`AdultStudents`).

# Vytvoření a odstranění

> [!success] Vytvoření
> ```sql
> CREATE VIEW AdultStudents AS
SELECT * FROM Students WHERE Age > 18;
> ```

> [!fail] Smazání
> ```sql
> DROP VIEW AdultStudents;
> ```

# TempDB

- Systémová databáze
- Slouží pro:
	- dočasné tabulky
	- [[#CTE - Common Table Expression|CTE]]
	- tabulkové proměnné
	- dočasné tabulky

# Typy dočasných tabulek v `tempdb`

## Lokální dočasná tabulka (`#`)

> [!success] Vytvoření
> ```sql
> CREATE TABLE #TempStudents (
>    Name NVARCHAR(50)
>);
> ```

- Viditelná jen v **aktuální relaci**
- Automaticky se maže po ukončení spojení

## Globální dočasná tabulka (`##`)

> [!success] Vytvoření
> ```sql
> CREATE TABLE ##AllStudents (
>    Name NVARCHAR(50)
>);
> ```

- Viditelná pro všechny uživatele
- Zanikne, až když ji nikdo nepoužívá

## Tabulková proměnná (`@`)

(technicky v tempdb, ale jiný princip)

> [!success] Vytvoření
> ```sql
>DECLARE @Students TABLE (
 >   Name NVARCHAR(50)
>);
> ```

# CTE - Common Table Expression

- Obecký tabulkový výraz
	- Dočasný výsledek jen pro jeden dotaz
	- Neukládá se
	- Zlepšuje čitelnost složitých dotazů

> [!success] Vytvoření
> ```sql
> WITH AdultStudents AS (
>    SELECT * FROM Students WHERE Age >= 18
>)
>SELECT * FROM AdultStudents;
>```

## Používá se hodně u:
- složitých [[Q15#Spojování tabulek (`JOIN`)|JOINů]]
- rekurze[^1] (např. stromové struktury)

---

# Tabulková proměnná

> [!success] Vytvoření
> ```sql
> DECLARE @Students TABLE (
> 	Id INT,
> 	Name NVARCHAR(50)
>);
>```

### Vlastnosti:
- Platí jen v rámci **aktuální dávky / procedury**
- Menší data
- Nelze na ně vytvářet indexy (kromě primárního klíče)
- Automaticky se zruší

# Rychlé srovnání

| Objekt               | Kde existuje | Jak dlouho             |
| -------------------- | ------------ | ---------------------- |
| Pohled               | Databáze     | Dokud ho nesmažu       |
| Dočasná tabulka `#`  | tempdb       | Do konce relace        |
| Dočasná tabulka `##` | tempdb       | Dokud ji někdo používá |
| CTE                  | Paměť        | Jeden dotaz            |
| Tabulková proměnná   | tempdb       | Aktuální dávka         |



---
[^1]: **Rekurzivní CTE** je CTE, které má **počáteční část** (start) a **rekurzivní část**, která se na CTE znovu odkazuje. Opakuje se, dokud už nenajde žádná další data
