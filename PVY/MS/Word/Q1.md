# Struktura wordovského dokumentu

| Název                    | Popis                                                            |
| ------------------------ | ---------------------------------------------------------------- |
| Znak                     | jednotlivé písmeno, číslice, symbol                              |
| Odstavec                 | text ukončený klávesou Enter                                     |
| Strana                   | jedna stránka dokumentu (ovlivněna okraji, orientací, velikostí) |
| Oddíl                    | část dokumentu s vlastním nastavením (jiné záhlaví, číslování)   |
| Dokument                 | celý soubor aplikace Word                                        |
| Záhlaví a zápatí         | horní a dolní část stránky (čísla stránek, název, datum)         |
| Členění textu do sloupců | rozdělení textu do více sloupců                                  |

# Práce s textem

**Základní úpravy formátování písma**

- typ písma (font)
- velikost písma
- **tučné**, *kurzíva*, podtržení
- barva písma
- zvýraznění textu
- horní/dolní index

# Vytváření a úprava tabulek

- tabulka je přehledné uspořádání řádků a sloupců
- můžeme upravovat:
	- velikost buněk
	- přidávat nebo mazat řádky a sloupce
	- spojovat a rozdělovat buňky
	- ohraničení tabulky a barvu
	- zarovnávat texty v buňkách

> [!tip]- Vytvoření

# Možnosti formátování a úpravy objektů

## Změna velikosti a tvaru

- můžeme objekt: zvětšovat, zmenšovat, otáčet, oříznout

## Formátování vzhledu objektu

- můžeme měnit: 
	- barvy
	- styl 
	- tloušťku obrysu
	- přidávat efekty
	- nastavit průhlednost

## Úpravy obrázku

- úprava jasu a kontrastu
- barevné korekce a filtry
- umělecké efekty
- odstranění pozadí

## Umístění objektu v dokumentu

- přesunutí objektu na stránce
- zarovnání objektu vůči stránce nebo jiným objektům
- přesné nastavení pozice objektu

## Seskupování objektů

- seskupení více objektů do jednoho celku
- společná úprava velikosti, pozice a vzhledu
- rozdělení seskupených objektů
- pro seskupení objektů stačí zakliknout jeden objekt a pak držet CTRL a zakliknout druhý objekt , a vyhledat seskupení objektů

# Styl

- je předem definovaný soubor formátovacích vlastností, který určuje vzhled textu nebo objektů v dokumentu
- pomocí stylu se zajistí jednotný vzhled dokumentu a snadné formátování

## Typy stylů:

| Styl           | Popis                                            |
| -------------- | ------------------------------------------------ |
| Odstavce       | formátuje celý odstavec (např: Nadpis, Normální) |
| Znaku          | formátuje pouze vybranou část textu              |
| Seznamu        | určuje vzhled odrážek a číslování                |
| Tabulky        | nastavuje vzhled tabulek                         |
| Propojený styl | kombinuje styl odstavce i znaku                  | 

## Kde může být styl uložen

- v aktuálním dokumentu
- v šabloně dokumentu
- v globální šabloně
- ve vlastní šabloně

---

# Šablona

- je to předpřipravený dokument, který slouží jako vzor pro vytváření nových dokumentů
- umožňuje rychlé vytvoření dokumentu se stejným vzhledem a strukturou

## Co může být v šabloně uloženo

- styly textu, rozložení stránky, záhlaví a zápatí, předdefinovaný text, tabulky, obrázky, grafické prvky, makra a automatizace

---

# Hromadná korespondence

- je nástroj, který slouží k **vytváření většího množství stejných dokumentů**, které se liší pouze **některými údaji** (např: jméno, adresa, datum)

- používá se pro: dopisy, obálky, štítky, emaily

- díky tomu se: šetří čas, snižuje chybovost

## Použití:

1. Vytvoří se hlavní dokument *(např. dopis se stejným textem pro všechny příjemce)*
2. Připojí se zdroj dat *(např. tabulka v Excelu, databáze, seznam kontaktů)*
3. Do dokumentu se vloží slučovací pole *(např. jméno, příjmení, adresa)*
4. Provede se sloučení dokumentů
5. Vzniknou samostatné dokumenty pro každého příjemce

- vytváří se automaticky na základě použitých stylů nadpisů

	- aktualizuje se při změnách v dokumentu
