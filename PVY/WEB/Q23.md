# Selektory

Slouží k výběru konkrétních HTML prvků, které chceme v CSS upravit.

| Typ selektoru | Syntaxe | Popis |
| :--- | :--- | :--- |
| **Univerzální** | `*` | Vybírá úplně všechny prvky na stránce. |
| **Element** | `p`, `h1`, `div` | Vybírá všechny prvky daného typu (tagu). |
| **Třída (Class)** | `.moje-trida` | Vybírá prvky s daným atributem `class`. Lze použít opakovaně. |
| **ID** | `#moje-id` | Vybírá prvek s unikátním `id`. Smí být na stránce jen jednou. |
| **Atribut** | `[type="text"]` | Vybírá prvky podle přítomnosti nebo hodnoty atributu. |

### Kombinace selektorů
Můžeme kombinovat více podmínek najednou:
- `p.text` – Vybere pouze odstavce, které mají třídu "text".
- `.btn.active` – Vybere prvky, které mají obě třídy zároveň.

### Vztahové selektory (Hierarchie)
- **Potomek:** `div p` – Vybere všechny odstavce kdekoli uvnitř `div`.
- **Přímý potomek:** `div > p` – Vybere pouze odstavce, které jsou přímými dětmi `div`.
- **Soused:** `h1 + p` – Vybere první odstavec, který následuje hned za `h1`.
- **Obecný sourozenec:** `h1 ~ p` – Vybere všechny odstavce, které jsou sourozenci `h1` (následují po něm).

# Pseudotřídy a Pseudoelementy

> [!info] Pseudotřídy (Stavy prvků)
> Reagují na stav prvku (např. interakci uživatele).
> - `:hover` – Při přejetí myší.
> - `:active` – Při kliknutí (stisknutí).
> - `:focus` – Když je prvek vybraný (např. aktivní input).
> - *(Další viz [[Q20#Stylování tabulek v CSS|otázku č. 20]])*

> [!info] Pseudoelementy (Části prvků)
> Stylují konkrétní část prvku bez změny HTML. Píší se se dvěma dvojtečkami `::`.
> - `::before` / `::after` – Vloží virtuální obsah před/za obsah prvku (vyžaduje vlastnost `content`).
> - `::first-letter` – Upraví první písmeno (např. iniciála v článku).
> - `::first-line` – Upraví první řádek textu.
> - `::selection` – Styluje text, který uživatel označil myší.
> - `::placeholder` – Styluje nápovědu uvnitř inputu.

# Priorita CSS stylů (Specificita)

Určuje, které pravidlo vyhraje, pokud na prvek působí více stylů najednou. Počítá se jako čtyři čísla `(a, b, c, d)`:

| Úroveň | Selektor | Hodnota (a, b, c, d) |
| :--- | :--- | :--- |
| **a** | Inline styl (přímo v HTML atributu `style`) | `1, 0, 0, 0` |
| **b** | ID selektory (`#header`) | `0, 1, 0, 0` |
| **c** | Třídy, atributy, pseudotřídy (`.btn`, `[type]`, `:hover`) | `0, 0, 1, 0` |
| **d** | Elementy a pseudoelementy (`p`, `h1`, `::before`) | `0, 0, 0, 1` |

*Příklad:* `nav ul li.active` má specificitu `0, 0, 1, 3` (jedna třída + tři elementy). Vyšší číslo v levější pozici vyhrává.

# Stylování textu a prvků

### Text
- `color` – Barva textu.
- `font-size` – Velikost písma.
- `font-family` – Typ písma (font).
- `font-weight` – Tloušťka (např. `bold`).
- `text-align` – Zarovnání (left, center, right, justify).
- `text-decoration` – Dekorace (např. `underline`).

### Ostatní (viz [[Q20#Stylování pozadí v CSS|otázku č. 20]])
- **Pozadí:** `background-color`, `background-image`.
- **Box Model:** `margin` (vnější odsazení) a `padding` (vnitřní odsazení).

# Responsivita (Responsivní web)

Cílem je, aby byl web čitelný a ovladatelný na všech zařízeních (mobil, tablet, desktop).

### Strategie návrhu
1. **Mobile-first:** Nejdříve navrhujeme pro mobil (jednodušší struktura). Styly pro větší obrazovky přidáváme postupně. Dnes preferovaný přístup kvůli výkonu.
2. **Desktop-first:** Nejdříve návrh pro PC, pak se odebírají nebo upravují prvky pro menší displeje.

### Media Queries (@media)
Umožňují aplikovat CSS pouze při splnění podmínek (velikost obrazovky).
- `min-width` – Styl platí **od** určité šířky nahoru.
- `max-width` – Styl platí **do** určité šířky dolů.

```css
/* Základní styl pro mobily */
body { background-color: white; }

/* Pro tablety a větší (nad 768px) */
@media (min-width: 768px) {
    body { background-color: lightgrey; }
}

/* Pro velké monitory (nad 1200px) */
@media (min-width: 1200px) {
    body { background-color: grey; }
}
```

### Absolutní a relativní jednotky
*(Podrobně zpracováno v [[Q20#CSS jednotky|otázce č. 20]])*
- **Absolutní (px):** Pevná velikost, nemění se.
- **Relativní (em, rem, %, vh, vw):** Přizpůsobují se kontextu (velikosti písma rodiče nebo velikosti okna). Klíčové pro responsivitu.
