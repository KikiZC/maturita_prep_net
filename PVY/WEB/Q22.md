# Význam a použití formulářů

Formuláře jsou základním prostředkem pro **komunikaci mezi uživatelem a webovou aplikací**.

- **Sběr dat:** Registrace, přihlášení, objednávky, kontaktní formuláře.
- **Zpracování:** Propojují klientskou stranu (Frontend) se serverem (Backend).
- **Interakce:** Umožňují zadávání, odesílání a následné ukládání dat do databáze.
- **Validace:** Kontrola správnosti zadaných údajů už před odesláním.

---

# Element `<form>`

Základní kontejner, který ohraničuje část stránky určenou pro odesílání dat. Bez tohoto elementu nelze formulář standardně odeslat na server.

### Základní atributy:
| Atribut | Význam |
| :--- | :--- |
| `action` | URL adresa, kam se data pošlou (relativní nebo absolutní cesta). |
| `method` | Určuje způsob odesílání dat (`GET` nebo `POST`). |
| `enctype` | Způsob kódování dat (důležité pro nahrávání souborů). |
| `name` | Název formuláře, důležitý pro identifikaci v JavaScriptu. |
| `novalidate` | Vypne vestavěnou HTML validaci prohlížeče. |
| `autocomplete` | Zapíná/vypíná našeptávání uložených hodnot. |
| `target` | Určuje, kde se otevře odpověď ze serveru (`_self`, `_blank`, `_parent`, `_top`). |

### Metody odesílání (GET vs. POST)
- **GET:** Data se přidají přímo do URL adresy (viditelné v řádku prohlížeče). Omezená délka, nepoužívat pro citlivá data. Vhodné pro vyhledávání.
- **POST:** Data se posílají v těle HTTP požadavku. Nejsou vidět v URL, nemají omezení délky. **Nutné pro hesla a citlivé údaje.**

### Typy kódování (enctype)
- `application/x-www-form-urlencoded`: Výchozí. Data jako `klíč=hodnota`.
- `multipart/form-data`: **Nutné pro nahrávání souborů** (`<input type="file">`).
- `text/plain`: Odesílá čistý text (málo používané).

# Formulářové prvky

### `<label>` a jeho propojení
Každý prvek by měl mít svůj popis. Propojení se dělá pomocí atributu `for` (u labelu) a `id` (u inputu).
```html
<label for="username">Uživatelské jméno:</label>
<input type="text" id="username" name="user">
```

### Typy elementu `<input>`
| Typ | Použití |
| :--- | :--- |
| `text` | Standardní jednořádkové textové pole. |
| `password` | Skryté znaky pro heslo. |
| `email` | Automatická kontrola formátu (vyžaduje `@`). |
| `number` | Povolí pouze číselný vstup. |
| `checkbox` | Zaškrtávací políčko (více možností). |
| `radio` | Výběr jedné možnosti ze skupiny (musí mít stejné `name`). |
| `file` | Nahrávání souborů ze zařízení. |
| `date` / `time` | Výběr data / času pomocí kalendáře/hodin. |
| `color` | Výběr barvy z palety. |
| `range` | Posuvník pro výběr hodnoty v rozsahu. |
| `tel` | Pro telefonní čísla (na mobilu vyvolá numerickou klávesnici). |
| `submit` | Tlačítko pro odeslání formuláře. |
| `reset` | Tlačítko pro vymazání všech polí do výchozího stavu. |

### Další prvky
- **`<select>` & `<option>`:** Rozbalovací seznam (dropdown).
- **`<textarea>`:** Víceřádkové pole pro delší texty.
- **`<button>`:** Univerzální tlačítko. Typy: `submit`, `reset`, `button`.
- **`<fieldset>` & `<legend>`:** Seskupení souvisejících prvků s nadpisem (vizuální rám).
- **`<datalist>`:** Našeptávač pro input.
- **`<optgroup>`:** Seskupení možností uvnitř `<select>`.

# Atributy polí (validace a stavy)

| Atribut | Význam |
| :--- | :--- |
| `name` | Název proměnné, která se odešle na server. |
| `value` | Výchozí nebo aktuální hodnota pole. |
| `placeholder` | Šedý nápovědný text uvnitř pole. |
| `required` | Pole musí být vyplněno pro odeslání. |
| `disabled` | Pole je neaktivní, zašedlé a **neodesílá se**. |
| `readonly` | Hodnotu nelze měnit, ale **odesílá se**. |
| `maxlength` | Maximální počet povolených znaků. |
| `min` / `max` | Číselné omezení rozsahu. |
| `checked` | Předem zaškrtnutý checkbox nebo radio button. |
| `pattern` | Validace pomocí regulárního výrazu (Regex). |

# Propojení s JavaScriptem

### Výběr elementů z DOM
```javascript
const form = document.getElementById('my-form');
const nameInput = document.querySelector('#name-input');
```

### Události (Events)
- `submit`: Spustí se při odeslání formuláře.
- `input`: Spustí se okamžitě při každé změně hodnoty v poli.
- `change`: Spustí se, když se hodnota změní a pole ztratí fokus (kliknutí mimo).
- `click`: Reakce na kliknutí (např. u tlačítka).

### Zpracování dat (Příklad)
Základem je `event.preventDefault()`, který zastaví výchozí odeslání (přebliknutí stránky), abychom data mohli zpracovat v JS.

```javascript
form.addEventListener('submit', function(event) {
    event.preventDefault(); // Zastaví odeslání na server

    let name = nameInput.value; // Čtení hodnoty
    let age = parseInt(document.getElementById('age').value, 10); // Převod na číslo

    if (name === "") {
        alert("Zadejte jméno!");
    } else {
        console.log("Jméno:", name, "Věk:", age);
    }
});
```

### Validace na klientské straně
Slouží k okamžité zpětné vazbě uživateli. Šetří čas i výkon serveru.
```javascript
nameInput.addEventListener('input', function() {
    if (this.value.length < 3) {
        this.style.borderColor = "red";
    } else {
        this.style.borderColor = "green";
    }
});
```
