# HTML tabulky a jejich struktura

Tabulka se skládá z logických celků, které pomáhají prohlížečům (a čtečkám) pochopit význam dat.
- `<table>` – obalující prvek.
- `<thead>`, `<tbody>`, `<tfoot>` – hlavička, tělo a patička tabulky.
- `<tr>` (Table Row) – řádek tabulky.
- `<th>` (Table Header) – buňka hlavičky (automaticky tučná a vycentrovaná).
- `<td>` (Table Data) – běžná buňka.

``` html
<table>
  <thead>                 // Hlavička tabulky
	<tr>                  // Jedna řádka tabulky
      <th>Jméno</th>      // Buňka hlavičky
      <th>Příjmení</th>
      <th>Body</th>
    </tr>
  </thead>
  <tbody>                 // Tělo tabulky se záznami
    <tr>
	  <td>Kiki</td>       // Jeden záznam v jedné buňce
      <td>ZC</td>
      <td>100</td>
    </tr>
    <tr>
      <td>Jan</td>
      <td>Novák</td>
      <td>85</td>
    </tr>
    <tr>
      <td>Petr</td>
      <td>Svoboda</td>
      <td>92</td>
    </tr>
  </tbody>
  <tfoot>                 // Patička tabulky
    <tr>
      <td colspan="2">Průměr</td>
      <td>92,3</td>
    </tr>
  </tfoot>
</table>
```

# Stylování tabulek v CSS

### Rámečky a prázdné prostory
- `border-collapse: collapse;` – **Klíčová vlastnost.** Spojí dvojité rámečky buněk do jednoho tenkého.
- `border-spacing` – mezera mezi buňkami (funguje jen pokud není nastaven collapse).
- `padding` – vnitřní odsazení v buňkách (`th`, `td`).

### Barvy a zarovnání
- `text-align` – horizontální zarovnání (left, center, right).
- `vertical-align` – vertikální zarovnání (top, middle, bottom).
- `background-color` – barva pozadí buňky nebo celého řádku.

### Pseudotřídy pro efektivní design
- `:nth-child(even/odd)` – vytvoření "zebry" (střídání barev řádků).
- `:hover` – zvýraznění řádku/buňky při přejetí myší.

> [!example] Příklad stylování
> ```css
> table { 
> 	width: 100%; 
> 	border-collapse: collapse; 
> }
> 
> th, td { 
> 	border: 1px solid black; 
> 	padding: 8px; 
> }
> 
> tr:nth-child(even) { 
> 	background-color: #f2f2f2; 
> }
> tr:hover { 
> 	background-color: #ddd; 
> }
> ```

# CSS jednotky

### 1. Absolutní jednotky
Mají pevnou velikost, která nezávisí na okolí.
- `px` (pixely) – základní jednotka pro web (vychází z rozlišení obrazovky).
- `pt` (body), `cm`, `mm` – používají se primárně pro **tiskové styly**.

### 2. Relativní jednotky
Velikost se mění podle kontextu (velikost písma, velikost okna).
- `%` – procenta z rozměru rodičovského prvku.
- `em` – násobek velikosti písma **daného prvku**.
- `rem` (root em) – násobek velikosti písma **kořenového prvku (`<html>`)**. (Doporučeno pro přístupnost).
- `vw` / `vh` (viewport width/height) – 1 % šířky nebo výšky okna prohlížeče.
- `vmin` / `vmax` – menší nebo větší rozměr z šířky/výšky okna.

# Stylování pozadí v CSS

Pozadí lze definovat buď jednotlivě, nebo zkratkou `background`.

- `background-color`: barva (název, HEX, RGB, HSL).
- `background-image: url('obrazek.jpg');`: cesta k souboru.
- `background-repeat`: `repeat`, `no-repeat`, `repeat-x` (vodorovně), `repeat-y` (svisle).
- `background-position`: umístění (např. `center center`, `top left`, `50% 50%`).
- `background-attachment`: 
    - `scroll` (hýbe se s obsahem),
    - `fixed` (zůstane stát při skrolování – efekt paralaxy).
- `background-size`: velikost obrázku (viz níže).

``` css
/* --- Souhrnný příklad vlastností pozadí --- */

.sekce-pozadi {
    /* 1. Barva pozadí (Název, HEX, RGB, HSL) */
    background-color: #3498db; /* Modrá barva pomocí HEX kódu */

    /* 2. Obrázek na pozadí */
    background-image: url('pozadi-krajina.jpg');

    /* 3. Opakování obrázku */
    /* no-repeat (jednou), repeat (dláždění), repeat-x (vodorovně), repeat-y (svisle) */
    background-repeat: no-repeat;

    /* 4. Pozice obrázku */
    /* Možnosti: center, top, bottom, left, right nebo souřadnice (např. 50% 50%) */
    background-position: center center;

    /* 5. Přichycení pozadí */
    /* scroll (standardní), fixed (obrázek stojí, obsah se sune - paralaxa) */
    background-attachment: fixed;

    /* 6. Velikost obrázku */
    /* cover (vyplní vše, ořízne), contain (vleze se celý, můžou být mezery), 100% 100% */
    background-size: cover;
}

/* --- ZKRÁCENÝ ZÁPIS (Shorthand) --- */
/* Pořadí: color image repeat attachment position / size */
.zkratka {
    background: #3498db url('obrazek.jpg') no-repeat fixed center / cover;
}
```

# Hodnoty Cover a Contain

Zásadní vlastnosti pro responzivní pozadí:

1. **`cover`**: Obrázek se roztáhne tak, aby **zcela pokryl** plochu prvku. Může dojít k oříznutí okrajů obrázku, ale nezůstane žádné prázdné místo.
2. **`contain`**: Obrázek se změní tak, aby se **celý vešel** do prvku. Zůstane vidět celý, ale mohou vzniknout prázdné plochy kolem (pokud má prvek jiný poměr stran).

# Pokročilé techniky pozadí

### Více pozadí
V CSS lze na jeden prvek vrstvit více obrázků/barev. Oddělují se čárkou (první v seznamu je nejvýše).
```css
background: url(logo.png) no-repeat top left, url(pozadi.jpg) cover;
```

### Gradienty (Přechody)

Gradient je v CSS brán jako obrázek (`image`).
- **Lineární**: `linear-gradient(to right, red, blue);`
- **Radiální**: `radial-gradient(circle, white, gray);`

### Průhlednost

- `opacity`: Nastaví průhlednost celému prvku **včetně jeho obsahu (textu)**.
- `rgba(r, g, b, a)`: Nastaví průhlednost **pouze barvě** (např. pozadí), text zůstane neprůhledný. (Hodnota `a` je 0 až 1).
