# Úvod do CSS Grid

> [!abstract] Co je to Grid?
> CSS Grid je moderní **dvourozměrný** systém rozložení prvků (layout), který umožňuje pracovat s řádky i sloupci zároveň. Je ideální pro celkovou strukturu stránky a komplexní rozvržení.

### Základní princip
1. Vytvoříme **Grid kontejner** (rodič) pomocí `display: grid;`.
2. Definujeme mřížku (sloupce a řádky).
3. Do mřížky vkládáme **Grid položky** (potomci).

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: grid; /* Zapne Grid */
  grid-template-columns: 1fr 1fr 1fr; /* 3 sloupce */
  gap: 10px; /* Mezery mezi boxy */
}
```

# Vlastnosti pro rodiče (Grid Kontejner)

Rodič určuje celkovou strukturu mřížky.

### Definice mřížky
- `grid-template-columns` – Definice sloupců.
  ```css
  grid-template-columns: 200px 1fr 200px; /* 3 sloupce: pevný, pružný, pevný */
  grid-template-columns: repeat(3, 1fr);  /* 3 stejně široké sloupce */
  ```
- `grid-template-rows` – Definice řádků.
  ```css
  grid-template-rows: 100px auto 100px; /* 3 řádky: pevný, podle obsahu, pevný */
  ```

### Jednotky a funkce
- **`fr` (fraction):** Zlomek dostupného prostoru.
  ```css
  grid-template-columns: 1fr 2fr; /* Levý zabere 1/3, pravý 2/3 šířky */
  ```
- **`auto`:** Velikost se přizpůsobí obsahu prvku.
- **`minmax(min, max)`:** Definuje rozsah velikosti.
  ```css
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); /* Responzivní mřížka */
  ```

### Mezery (Gap)
- `gap` – Společná mezera pro řádky i sloupce.
  ```css
  gap: 20px; /* Mezera 20px mezi všemi buňkami */
  ```

### Zarovnání celého Gridu (v kontejneru)
- `justify-content` – Vodorovné zarovnání celého gridu vůči rodiči.
- `align-content` – Svislé zarovnání celého gridu vůči rodiči.
  ```css
  justify-content: center; /* Vycentruje celou mřížku vodorovně */
  align-content: space-between; /* Roztáhne řádky k okrajům */
  ```
  *Hodnoty: `start`, `center`, `end`, `space-between`, `space-around`, `space-evenly`.*

# Automatické rozmísťování

Pokud neurčíme pozici, prohlížeč prvky skládá automaticky.

- **`grid-auto-flow`**:
    - `row` (výchozí) – Plní řádky.
    - `column` – Plní sloupce.
- **`grid-auto-rows` / `grid-auto-columns`**: Určuje velikost automaticky vytvořených řádků/sloupců (např. když do mřížky 2x2 vložíme 5. prvek).

# Grid vs. Flex a jiné techniky

| Vlastnost | CSS Grid | CSS Flexbox |
| :--- | :--- | :--- |
| **Rozměr** | 2D (řádky i sloupce zároveň) | 1D (pouze řada nebo sloupec) |
| **Použití** | Hlavní layout, galerie, mřížky | Navigace, drobné komponenty, centrování |
| **Princip** | Layout-first (definujeme mřížku) | Content-first (prvky se řadí za sebe) |

### Starší techniky
- **Float:** Původně pro obtékání obrázků, dříve zneužíván pro layout. Náročný na údržbu (nutnost "clearovat" floay).
- **Table layout:** Starý způsob pomocí HTML tabulek. Nepružné, sémanticky špatně.
- **Position:** Používá se pro překrývání prvků (`absolute`, `fixed`), ne pro celkový layout.
