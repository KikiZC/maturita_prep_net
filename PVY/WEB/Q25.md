# Úvod do CSS Flexbox

> [!abstract] Co je to Flexbox?
> Flexbox je moderní **jednorozměrný** layout systém. Pracuje vždy v jednom směru najednou – buď v řádku (`row`), nebo ve sloupci (`column`).

### Význam a využití
- **Jednoduchost:** Snadné zarovnání prvků horizontálně i vertikálně.
- **Flexibilita:** Prvky se umí automaticky smršťovat nebo natahovat podle místa.
- **Použití:** Navigační menu, centrování prvků, karty (cards), drobné komponenty.

# Flex Kontejner (Rodič)

Flexbox zapneme na rodičovském elementu pomocí `display: flex;`. Tím se z jeho přímých potomků stanou flex položky.

### Hlavní a křížová osa

Směr os se mění podle toho, co nastavíš ve `flex-direction`.

- **Hlavní osa (Main Axis):** Směr, kterým se prvky řadí. Zarovnáváme na ní pomocí `justify-content`.
- **Křížová osa (Cross Axis):** Směr kolmý na hlavní osu. Zarovnáváme na ní pomocí `align-items`.

### Vlastnosti kontejneru

#### 1. `flex-direction` (Směr hlavní osy)
```css
flex-direction: row;            /* Vedle sebe (výchozí) */
flex-direction: row-reverse;    /* Vedle sebe zprava doleva */
flex-direction: column;         /* Pod sebou shora dolů */
flex-direction: column-reverse; /* Pod sebou zdola nahoru */
```

#### 2. `justify-content` (Zarovnání na hlavní ose)
| Hodnota | Popis |
| :--- | :--- |
| `flex-start` | Prvky na začátku osy. |
| `flex-end` | Prvky na konci osy. |
| `center` | Prvky uprostřed. |
| `space-between` | Stejné mezery **mezi** prvky (první a poslední jsou u okrajů). |
| `space-around` | Stejné mezery **kolem** každého prvku. |
| `space-evenly` | Úplně stejné mezery všude (včetně okrajů). |

#### 3. `flex-wrap` (Zalamování)
Určuje, zda se prvky mají vejít na jeden řádek, nebo se mohou zalomit.
```css
flex-wrap: nowrap; /* Vše na jednom řádku (může přetéct) */
flex-wrap: wrap;   /* Zalomení na další řádky */
```

#### 4. `align-items` (Zarovnání na křížové ose)
Zarovnává **všechny** prvky v rámci jedné řady.
```css
align-items: stretch;  /* Výchozí, prvky se roztáhnou na celou výšku/šířku */
align-items: center;   /* Vycentrování */
align-items: flex-start; / flex-end; /* Začátek nebo konec */
align-items: baseline; /* Zarovnání podle účaří (textu) */
```

#### 5. `align-content` (Zarovnání řad)
Funguje pouze pokud je zapnutý `flex-wrap` a prvků je více řad. Zarovnává celé řady vůči kontejneru.
- Hodnoty jsou podobné jako u `justify-content` (`center`, `space-between` atd.).

# Flex Položky (Potomci)

Vlastnosti, které se nastavují přímo konkrétnímu prvku uvnitř flexboxu.

### Velikost a růst
- **`flex-grow`**: Určuje, jak moc má prvek vyrůst, aby vyplnil prázdné místo (výchozí `0`).
- **`flex-shrink`**: Určuje, jak moc se má prvek smrsknout, pokud není místo (výchozí `1`).
- **`flex-basis`**: Základní velikost prvku před rozdělením prostoru (např. `200px`).

### Zkratka `flex`
Doporučený zápis pro růst, smrštění a základ:
```css
flex: 1 1 200px; /* grow | shrink | basis */
flex: 1; /* Zkratka pro "vyplň zbytek místa" */
```

### Ostatní vlastnosti potomka
- **`order`**: Mění pořadí prvku bez ohledu na HTML (např. `order: -1` posune prvek na začátek).
- **`align-self`**: Přepíše `align-items` pro jeden konkrétní prvek.
  ```css
  .special-item { align-self: center; }
  ```

# Flex vs. Grid

| Vlastnost | CSS Flexbox | CSS Grid |
| :--- | :--- | :--- |
| **Rozměr** | 1D (Řada **nebo** sloupec) | 2D (Řada **i** sloupec zároveň) |
| **Zaměření** | Komponenty, menu, centrování | Hlavní layout, galerie, mřížky |
| **Princip** | Content-first (prvky tlačí na kontejner) | Layout-first (kontejner definuje mřížku) |

### Kombinace (Best Practice)
- **Grid** se použije pro **vnější strukturu** stránky (Header, Sidebar, Content).
- **Flex** se použije **uvnitř** těchto částí (např. pro zarovnání tlačítek v navigaci nebo ikon v postranním panelu).
