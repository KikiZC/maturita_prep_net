# 3.4 Kvadratické rovnice
[[../Matematika-Okruhy|⬅ Zpět na přehled okruhů]]

Kvadratická rovnice má obecný tvar $ax^2 + bx + c = 0$ ($a \neq 0$). Grafem je parabola.

## 1. Výpočet pomocí diskriminantu ($D$)

Nejbezpečnější cesta k výsledku.

**Vzorec pro diskriminant:**
$$D = b^2 - 4ac$$

**Diskuse řešení podle $D$:**
*   **$D > 0$:** 2 různé reálné kořeny.
*   **$D = 0$:** 1 dvojnásobný reálný kořen.
*   **$D < 0$:** Žádné reálné řešení ($K = \emptyset$).

**Vzorec pro výpočet kořenů ($x_{1,2}$):**
$$x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}$$

---

## 2. Neúplné kvadratické rovnice (Rychlá cesta)

Někdy nemusíš počítat diskriminant:
1.  **Chybí $c$ ($ax^2 + bx = 0$):** Vyřešíš vytýkáním $x$. Kořeny jsou $x_1 = 0$ a $x_2$ ze závorky.
    *   $x^2 - 5x = 0 \Rightarrow x(x - 5) = 0 \Rightarrow x_1 = 0, x_2 = 5$.
2.  **Chybí $b$ ($ax^2 + c = 0$):** Vyřešíš odmocněním.
    *   $x^2 - 9 = 0 \Rightarrow x^2 = 9 \Rightarrow x = \pm 3$.

---

## 3. Vietovy vzorce (Vztahy mezi kořeny a koeficienty)

Užitečné pro rychlý rozklad na součin (když $a = 1$):
*   $x_1 + x_2 = -b$
*   $x_1 \cdot x_2 = c$

---

## 4. Kvadratické nerovnice

Řešíš jako rovnici, najdeš kořeny a pak určíš interval (např. pomocí grafu paraboly nebo tabulky znamének).

> [!TIP]
> **Graf paraboly:**
> *   Pokud $a > 0$ (parabola se "směje"), je pod osou mezi kořeny.
> *   Pokud $a < 0$ (parabola se "mračí"), je nad osou mezi kořeny.
