# 5.1 Základní poznatky o posloupnostech
[[../Matematika-Okruhy|⬅ Zpět na přehled okruhů]]

Posloupnost je funkce, jejímž definičním oborem je množina přirozených čísel $\mathbb{N}$. Členy posloupnosti značíme $a_1, a_2, a_3, \dots, a_n$.

## 1. Zadání posloupnosti

1.  **Vzorcem pro n-tý člen:** Předpis, podle kterého vypočítáš libovolný člen.
    *   *Příklad:* $a_n = 2n + 1 \Rightarrow a_1 = 3, a_2 = 5, a_3 = 7, \dots$
2.  **Rekurentně:** Další člen se vypočítá pomocí těch předchozích. Musíš znát začátek.
    *   *Příklad:* $a_1 = 2, a_{n+1} = a_n + 3 \Rightarrow a_2 = 2+3=5, a_3 = 5+3=8, \dots$
3.  **Výčtem prvků:** $\{2, 4, 6, 8, \dots\}$
4.  **Graficky:** Grafem jsou **izolované body**. (Na ose $x$ jsou jen celá kladná čísla $1, 2, 3 \dots$).

---

## 2. Vlastnosti posloupností

Stejné jako u funkcí:
*   **Rostoucí:** $a_{n+1} > a_n$ (každý další člen je větší než ten předchozí).
*   **Klesající:** $a_{n+1} < a_n$.

> [!TIP]
> Pokud chceš vědět, jestli je posloupnost rostoucí, spočítej si první tři členy. Pokud rostou a vzorec je lineární, bude nejspíš rostoucí celá.
