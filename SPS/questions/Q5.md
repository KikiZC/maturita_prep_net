# IP adresa a její struktura

## IP adresa

je jedinečný identifikátor zařízení v síti

Máme dva typy:
- IPv4 která má 32 bitů
- IPv6 která má 128 bitů

### Složení

Adresa sítě - identifikuje konkrétní síť
Adresa uzlu – identifikuje konkrétní zařízení v síti

Rozdělení `194.24.245.30 /24`

| Síťová část | Host |
|:-----------:|:----:|
| 194.24.245. |  30  |

### Třídy IP adres


| Třída |  Začátek  |   Použití    |
|:-----:|:---------:|:------------:|
|   A   |  1 - 126  |  Velké sítě  |
|   B   | 128 - 191 | Střední sítě |
|   C   | 192 - 223 |  Malé sítě   |
|   D   | 224 - 239 |  Multicast   |
|   E   | 240 - 255 | Rezervováno  |

### Adresa broadcast

- slouží k odeslání zprávy všem zařízením v síti
- je to nejvyšší možná adresa v síti jako např.: `192.168.1.255`
- nevýhoda že zatěžuje síť

## Maska sítě

- určuje, která část IP adresy je síťová a která hostitelská
- jsou dva druhy zápisu:
	- desítkový: `255.255.255.0`
	- prefixový: `/24`
- **konstantní maska**: pevně daná (např. třída C má vždy /24)
- **variabilní maska**: umožňuje rozdělit síť na podsítě různých velikostí

## Dělba sítě na podsítě

- umožňuje rozdělit větší síť na menší části.
- Např. síť `192.168.1.0/24` lze rozdělit na:
	- 192.168.1.0/26 (64 adres)
	-192.168.1.64/26
	- 192.168.1.128/26
	- 192.168.1.192/26

(zbytek počítání by mělo být jasné)

### Zjištění informací o síti ve Windows

V CMD zadám tento příkaz

```
ipconfig /All
```

a to mi zobrazí IP adresu, masku podsítě, výchozí bránu

### Zjištění informací o síti v Linuxu

Buďto pomocí

```
ip a
```

 nebo
 
```
ifconfig
```
