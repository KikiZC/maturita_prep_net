# Služba FTP serveru v systému Linux

>[!question] Služba FTP serveru v systému Linux
>Instalace a konfigurace síťových služeb ftp, pasivní a aktivní ftp server; anonymní ftp server, osobní ftp servery uživatelů - zabezpečený přístup, zabezpečený přenos pomocí SSL mezi ftp serverem a ftp klientem) a ssh (instalace a konfigurace ssh, ověřování přístupu pomocí hesel či osobních certifikátů uživatelů).

---

## Definice

>[!info] FTP
>FTP = File Transfer Protocol
>Slouží k přenášení souborů po síti

>[!info] Aktivní/Pasivní FTP
>Aktivní
>
>- Server vytváří připojení ke klientovi
>
>Pasivní
>
>- Nástupce Aktivního FTP
>- Server pouze poslouchá a čeká na připojení od klienta
>- Firewall-friendly

>[!info] Anonymní FTP
>Na rozdíl od normálního FTP, které vyžaduje jméno/heslo, tak k anonymnímu FTP má přístup kdokoliv s adresou serveru

---

## Základní info

- **Šifrování**: Ne (jeho variantu můžou podporovat)
- **Porty**: 20 (přenost dat) a 21 (příkazy)
- **Transportní protokol**: TCP
